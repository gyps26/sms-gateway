Getting Started

  • Log in to your text local account and navigate to Settings → API Keys.

  • Click on the Create New Key button to add a new API key.

  • Copy the newly created API key.

Configurations

API Key

Enter the API Key you got from the Textlocal.

URL

Provide the endpoint URL (e.g., https://api.textlocal.in/) for the Textlocal.
