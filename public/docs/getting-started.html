Getting Started

This app allows you to send SMS messages and USSD Pull requests directly through your Android device.

How It Works

  • Install the companion Android app on your phone.

  • Log in using your web app credentials to link your device.

  • When you send a message or USSD Pull request from the web app, it is instantly synced with the mobile app and dispatched using your phone.

Features

  • Global Messaging: Send SMS or MMS to any number worldwide, as long as your phone supports sending to that destination.

  • Message Syncing: Incoming SMS or MMS messages on your phone are automatically synced to the web app.

  • Carrier Charges: Standard SMS or MMS charges from your mobile network provider will apply.

Optional: Use External Gateways

You can also choose to send messages via third-party gateways like Textlocal, Twilio, and others by adding them as Sending Server. If you use one of these services, the Android app is not required.
