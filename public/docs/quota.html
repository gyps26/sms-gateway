Quota

You can set Quota for SIM and Sending Server. This is useful to prevent sending more messages than the rate limit allows. If the number of messages exceeds the allowed quota, they will be scheduled for sending after the quota resets.

Enabled

Quota will be in effect if it is enabled, otherwise it won't limit anything.

Value

Number of messages allowed to send.

Frequency and Frequency Unit

This defines the time frame for the quota.

Reset At

This field is optional. The system resets the quota at the given date and time. If not specified, it will be calculated by adding the frequency to the current date and time.

Timezone

This is required when Reset At is set. Reset At datetime will be calculated according to this timezone.
