Sending Servers

You can use third party SMS Gateways to send messages. You have to add Sending Servers for that. You can go to Sending Servers by clicking on it from the menu and clicking on the Add button.

Fields

Name

Enter a name for the sending server.

Driver

Select the SMS provider (e.g., Textlocal) from the dropdown list.

Supported Types

Check the types of messages you want to send using this Sending Server.

Enabled

Toggle whether the sending server is active.

Configuration

This is where you enter the configuration for this specific driver. You can check out Custom, Textlocal or Twilio for that.
