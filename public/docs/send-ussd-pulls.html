Send USSD Pulls

Click on USSD Pulls from the navigation menu and click on the Send button from the top right to start sending messages.

Fields

Name

Name of the campaign. It is an optional field, but it will help you keep track of the campaign on the Campaigns list.

SIMs

SIMs you want to use for the campaign. It will split USSD Pulls equally if multiple SIMs are selected.

Codes

USSD codes you want to send one per line.

Delay

Delay between each USSD pull request when sending multiple requests. It should be between 0 and 120. You can also use a range delay like 10–60, and it will select a random delay between 10 and 60 for each request. It is advisable to maintain a delay of 30 to 60 seconds between each USSD pull request. Since a USSD pull session can last up to 60 seconds, sending a new request while the previous session is still active may cause the new request to be treated as input for the ongoing session, leading to a failure.

Prioritize

Allows you to choose if you want to send the campaign in a prioritized queue. If you select Yes, then it will postpone any non-prioritized campaign currently running and send this campaign first.

Schedule

Date and Time when you want to send this campaign.

Recurring

Lets you choose if you want to repeat the campaign after a certain time frame or not.

Frequency

Only available when you choose to repeat the campaign. It allows you to specify the frequency of the recurring campaign. For example, if you select 1 Month, then it will repeat the campaign every month.

Ends At

Only available when you choose to repeat the campaign. Set the date and time when you want to stop repeating the campaign.

Active Hours

Time of the day when USSD Pulls should be sent. If there are still USSD Pulls left to send in the campaign after this time, then it will schedule them for the next available day and time.

Timezone

Timezone used by Active Hours, Days of Week, Schedule and Ends At.

Days of Week

Days of the week when the campaign should be active. If the current day doesn't fall within the selected days, then it will schedule remaining USSD Pulls for the next available day and time.
