Devices

You can add an Android device to your account. This will also list all the SIMs connected to the device on the SIMs page. You will be able to select SIMs while sending messages or USSD Pull requests.

Installation

Follow these steps to install the app on your Android device manually:

1. Download the APK File

Download the APK file from a Download link shown after clicking Add on the Devices page in the web app.

2. Enable installation from unknown sources

Depending on your Android version, follow the instructions below:

For Android 8.0 (Oreo) and above:

  1. Open Settings on your phone.

  2. Navigate to Apps > Advanced > Special app access > Install unknown apps.

  3. Select the browser or file manager you are using to download the APK.

  4. Toggle on Allow from this source.

For Android 7.0 (Nougat) and earlier:

  1. Open Settings.

  2. Go to Security.

  3. Enable Unknown sources by toggling it on.

  4. Confirm the warning message by tapping OK.

3. Locate the APK File

Use your file manager app to find the downloaded APK file (usually in the Downloads folder).

4. Install

Tap on the APK file to begin the installation and tap Install. If you get App blocked to protect your device alert, then follow steps from here to resolve the issue.

5. Open

Once the installation is complete, tap Open to launch the app.

6. Sign In

Sign in using your web app credentials to connect the app to the SMS Gateway.

OR

Tap on "Sign in using QR code" and scan the QR code shown after clicking Add on the Devices page in the web app.

Note: Ensure your phone is connected to the internet during the login process.

That's it! Your app should now be ready to use.
