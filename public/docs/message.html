Message

It can be an SMS, MMS or WhatsApp message. It can have the following status values.

Status

Pending

It is not yet picked up by the Device or Sending Server for sending.

Queued

It is queued for sending on Device or Sending Server.

Processed

Device or Sending Server processed the message successfully. This indicates that Device or Sending Server done its job, and it is now up to the operator to send it further.

Sent

This indicates that the message was sent.

Delivered

This indicates that the message was delivered. This status is possible only if the delivery report for the campaign was enabled.

Failed

This indicates that there was a failure to send the message.

Received

This indicates that this message was received.
