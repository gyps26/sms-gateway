User Help

Cancel

You can cancel the campaign at any time. This will stop the campaign and prevent any further messages or USSD Pulls from being sent.

Canceling a campaign while it is sending will not stop it immediately. It will take some time for the system to process the cancellation. If you want to cancel a campaign on a Device immediately, then it is recommended to stop it from the Device itself by using the Cancel button on notification.

How to Cancel

  1. Go to the Campaigns section in the app.

  2. Find the campaign you want to cancel.

  3. Click on the Cancel button next to the campaign.

  4. Confirm the cancellation when prompted.

You can only cancel campaigns that are currently in Processed or Scheduled status.

Partial Cancellation

If a campaign was sent using multiple Devices or Sending Servers, you can cancel it partially. This means you can stop sending messages or USSD Pulls from specific Devices or Sending Servers while allowing others to continue.

  1. Go to the Campaigns section in the app.

  2. Find the campaign you want to cancel partially.

  3. Click on the Devices or Sending Servers button next to the campaign.

  4. Find the Device or Sending Server you want to cancel.

  5. Click on the Cancel button next to the Device or Sending Server.

You can only cancel Devices or Sending Servers that are currently in Pending, Queued or Stalled status.
